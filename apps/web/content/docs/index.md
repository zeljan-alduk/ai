ALDO AI is the LLM-agnostic AI sub-agent orchestrator. Define agents
as YAML, route to any model, swap cloud and local without touching
code, replay every run, and enforce privacy at the platform layer.

These docs are organised into five sections:

- **Quickstart** — sign up, seed the default agency, run your first agent.
- **Concepts** — mental models that shape every part of the platform.
- **Guides** — task-oriented walkthroughs for common workflows.
- **API reference** — auto-generated from the Zod contract package.
- **SDKs** — TypeScript and Python clients.

If you read only one page, read the
[Quickstart](/docs/quickstart). If you read two,
follow it with
[Capability-class routing](/docs/concepts/capability-class-routing).
